Perímetro de un Trapecio

Para calcular el perímetro necesitas ingresar los lados c y d que se muestran en la figura y las bases del trapecio.











Diagonal de un Trapecio
En un trapecio se pueden encontrar 2 diagonales, para cada diagonal se usa una misma formula pero se usan diferentes datos.

Para calcular la primera diagonal se necesita la base mayor, el lado d y el ángulo en grados β.

Para calcular la segunda diagonal se necesita la base mayor, el lado c y el ángulo en grados α.
