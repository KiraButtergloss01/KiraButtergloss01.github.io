Área de un Triángulo
En este apartado se calcula el área de un Triángulo cuya base es denotada por B y su altura por h. Esta calculadora funciona para triangulos escalenos, obtusángulos y rectángulos
En el caso del Triángulo rectángulo se usan los catetos denotados por a y b como se aprecia en las figuras.














Área de un Triángulo equilátero
En este apartado se calcula el área de un Triángulo equilátero cuyos lados están denotados por x.












Área de un Triángulo isóceles
En este apartado se calcula el área de un Triángulo isóceles cuyos lados están denotados por a y L y su altura por h como se muestra en la figura.
¿No conoces la altura?, No hay problema deja el espacio en blanco y la calculadora la encontrará por ti.
















¿No sabes que tipo de triángulo es?
No te procupes, este apartado se calcula el área para cualquier tipo de triángulo utilizando la fórmula de Herón Para usarla lo único que necesitas es conocer la medida de todos las lados del triángulo. Dichos lados estarán denotados por a, b y su c como se muestra en la figura.
